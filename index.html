Controllers

 
// Controller names are Pascal Case
app.controller('MyController', function() {
  // data and behavior go in here, NOT in the view
  this.items = ['first', 'second', 'third'];

  this.addItem = function(newItem) {
    this.items.push(newItem);
  };
});
  • Should be short - there are other places to put logic
  • In particular, should NOT manipulate the dom
  • Injectable (as are most things in Angular)
