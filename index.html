Controllers

 
// Controller names are Pascal Case
app.controller('MyController', function() {
  // data and behavior go in here, NOT in the view
  this.items = ['first', 'second', 'third'];

  this.addItem = function(newItem) {
    this.items.push(newItem);
  };
});
  • Control some part of the DOM
  • *Open the demo
  • Should be short - there are other places to put logic
  • Should NOT manipulate the dom
  • Injectable (as are most things in Angular)

DI

Inject all the things
Simple

app.service('myService', function($http) {
  this.getStuff = function() {
    return $http.get('some/url');
  };
});

app.controller('MyController', function($scope, myService) {
  myService.getStuff().success(function(data) {
    $scope.stuff = data;
  });
});
  • $ prefix indicates Angular builtin service
